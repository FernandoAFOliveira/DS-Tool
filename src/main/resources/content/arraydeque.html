ArrayDeque

ArrayDeque is a resizable double-ended queue. It is useful when you need efficient insertion and removal from both the front and back. Use ArrayDeque for queues, stacks, undo systems, or processing items from both ends.

Key idea: ArrayDeque = Stack + Queue in one structure.

API Overview

addFirst(E e), addLast(E e), pollFirst(), pollLast(), peekFirst(), peekLast(), size()

Code

ArrayDeque<Integer> dq = new ArrayDeque<>();

// --- Queue behavior (FIFO) ---
dq.addLast(1);
dq.addLast(2);
dq.addLast(3);

System.out.println(dq.pollFirst()); // 1

// --- Stack behavior (LIFO) ---
dq.push(10);   // same as addFirst
dq.push(20);

System.out.println(dq.pop()); // 20

// --- Peek without removing ---
System.out.println(dq.peekFirst()); // front
System.out.println(dq.peekLast());  // back

// --- Iteration ---
for (Integer x : dq) {
    System.out.println(x);
}
