HashMap

HashMap stores key-value pairs and provides very fast average lookup, insertion, and deletion by key. Use HashMap for counting words, storing usernames with scores, or mapping IDs to objects.

API Overview

put(K key, V value), get(Object key), containsKey(Object key), remove(Object key), keySet(), size()

Key idea: Use entrySet() to iterate efficiently over keys and values. Avoid looping over keySet() and calling get() repeatedly.

Code

HashMap<String, Integer> wordCount = new HashMap<>();

wordCount.put("hello", 1);
wordCount.put("world", 2);

// Access
System.out.println(wordCount.get("hello")); // 1
System.out.println(wordCount.containsKey("world")); // true

// Update value
wordCount.put("hello", wordCount.get("hello") + 1); // increment count

// Correct iteration (best practice)
for (Map.Entry<String, Integer> entry : wordCount.entrySet()) {
    System.out.println(entry.getKey() + " -> " + entry.getValue());
}
