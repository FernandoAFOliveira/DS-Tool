TreeSet

TreeSet stores unique elements in sorted order and supports navigation methods for finding nearby values. Use TreeSet when you need unique values, sorted order, or operations like finding the next higher or lower value.

Key idea: Sorted + navigation (find neighbors quickly)

API Overview

add(E e), contains(Object o), remove(Object o), first(), last(), lower(E e), higher(E e), floor(E e), ceiling(E e)

Navigation:
lower(x) → strictly less than x
floor(x) → ≤ x
higher(x) → strictly greater than x
ceiling(x) → ≥ x
Important: If the value exists: floor(x) = x and ceiling(x) = x.

Code

TreeSet<String> set = new TreeSet<>();

set.add("banana");
set.add("apple");
set.add("cherry");

// Sorted automatically: [apple, banana, cherry]

System.out.println(set.first());   // apple
System.out.println(set.last());    // cherry

System.out.println(set.lower("banana"));   // apple
System.out.println(set.floor("banana"));   // banana

System.out.println(set.higher("banana"));  // cherry
System.out.println(set.ceiling("banana")); // banana
