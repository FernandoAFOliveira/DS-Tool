Stack

Stack represents last-in, first-out behavior, where the most recently added item is removed first. Use stack behavior for undo operations, expression evaluation, recursion simulation, or backtracking.

API Overview

push(E e), pop(), peek(), empty(), search(Object o)

Key idea: Stack = Last-In, First-Out (LIFO). The most recently added element is removed first. In modern Java, ArrayDeque is preferred over Stack.
Top → [first, second, third] Only the top element is accessible.

Code

// Legacy Stack (works, but not recommended)
Stack<String> stack = new Stack<>();

stack.push("first");
stack.push("second");

System.out.println(stack.pop());  // second
System.out.println(stack.peek()); // first


// Recommended: use ArrayDeque as a stack
ArrayDeque<String> dq = new ArrayDeque<>();

dq.push("first");
dq.push("second");

System.out.println(dq.pop());  // second
System.out.println(dq.peek()); // first
