ArrayList

Resizable array implementation of the List interface. Provides fast random access but slower insertions and deletions in the middle. Use ArrayList when storing items where you often access elements by index, such as a list of users, products, or scores.

API Overview

add(), get(), set(), remove(), size(), contains()

Key idea: ArrayList gives fast index access, but removing items shifts later elements left. Use an Iterator when removing during traversal.

Code

ArrayList<Integer> numbers = new ArrayList<>();

numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.add(4);
numbers.add(5);

// Index access
System.out.println(numbers.get(0)); // 1

// Safe removal while iterating
Iterator<Integer> iter = numbers.iterator();

while (iter.hasNext()) {
    int value = iter.next();

    if (value % 2 == 0) {
        iter.remove();
    }
}

System.out.println(numbers); // [1, 3, 5]
