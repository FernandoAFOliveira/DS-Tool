ArrayList

Description

Resizable array implementation of the List interface. Provides fast random access but slower insertions and deletions in the middle.

Example

Use ArrayList when storing items where you often access elements by index, such as a list of users, products, or scores.

API Overview

add(), get(), set(), remove(), size(), contains()

Code

ArrayList<String> list = new ArrayList<>();

list.add("Alice");
list.add("Bob");

System.out.println(list.get(0)); // Alice

list.remove("Bob");
System.out.println(list.size()); // 1
