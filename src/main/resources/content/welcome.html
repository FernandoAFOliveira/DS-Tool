Welcome to Data Structure Advisor

This app helps students and developers compare common Java data structures based on the needs of a programming task.

How to use it

Use the questions on the left to describe what your program needs. The data structure list will update as you choose options.

Click a data structure name to view its diagram, explanation, common methods, and example code.

Tip

Choose Any when a feature does not matter for your problem. Use the compact controls to rank lookup speed, add/delete speed, and memory efficiency.
