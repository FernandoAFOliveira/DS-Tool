Questions Panel

The questions help the app understand what kind of data structure your program needs.

Yes / No / Any Questions

Use these options when a requirement is either needed, not needed, or does not matter.

  • Yes means the data structure must support that feature.
  • No means the data structure should not use that feature.
  • Any means the feature should not affect the recommendation.

Priority Controls

The number controls let you decide how important each performance category is. Higher numbers give that category more influence in the ranking.

  • Lookup speed prioritizes fast searching or access.
  • Add/Delete speed prioritizes fast insertion and removal.
  • Memory efficiency prioritizes lower memory overhead.

Data Structure List

The list updates automatically. Higher-ranked structures appear closer to the top. Select a structure to see its diagram and full explanation.
