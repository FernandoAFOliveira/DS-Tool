Removal order

Choose how items should be removed from the structure.

  • Any – Order does not matter
  • FIFO – First In, First Out (Queue behavior)
  • LIFO – Last In, First Out (Stack behavior)
  • Double Ended – Efficient add/remove from both ends (ArrayDeque)
  • Priority – Removal based on priority (PriorityQueue)

PriorityQueue typically removes the smallest element first unless a custom comparator is used.
