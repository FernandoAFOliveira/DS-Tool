TreeMap

TreeMap stores key-value pairs in sorted key order and supports navigation methods such as finding nearby keys. Use TreeMap when you need key-value mapping and also need the keys kept sorted.

API Overview

put(K key, V value), get(Object key), containsKey(Object key), firstKey(), lastKey(),

Key idea: TreeMap = HashMap + sorting (by keys). Keys are always kept in sorted order, not insertion order. Important: Keys must be comparable (or use a Comparator), or a runtime error will occur.
Navigation:
lowerKey(x) → strictly less than x
floorKey(x) → ≤ x
higherKey(x) → strictly greater than x
ceilingKey(x) → ≥ x

Code

TreeMap<String, Integer> map = new TreeMap<>();

map.put("banana", 2);
map.put("apple", 1);
map.put("cherry", 3);

// Sorted by key: {apple=1, banana=2, cherry=3}

System.out.println(map.firstKey()); // apple
System.out.println(map.lastKey());  // cherry

// Navigation
System.out.println(map.lowerKey("banana"));   // apple
System.out.println(map.floorKey("banana"));   // banana
System.out.println(map.higherKey("banana"));  // cherry
System.out.println(map.ceilingKey("banana")); // banana

// Correct iteration (like HashMap)
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " -> " + entry.getValue());
}
