HashSet

HashSet stores unique elements and provides fast average add, remove, and contains operations. Use HashSet when you need to remove duplicates or quickly check whether an item has already appeared.

API Overview

add(E e), contains(Object o), remove(Object o), clear(), size()

Code

HashSet<Integer> a = new HashSet<>();
HashSet<Integer> b = new HashSet<>();

a.add(1);
a.add(2);
a.add(3);

b.add(3);
b.add(4);

// Iterate through a HashSet
for (Integer x : a) {
    System.out.println(x);
}

// Union: all values from a or b
HashSet<Integer> union = new HashSet<>(a);
for (Integer x : b) {
    union.add(x);
}

// Intersection: values found in both a and b
HashSet<Integer> intersection = new HashSet<>();
for (Integer x : a) {
    if (b.contains(x)) {
        intersection.add(x);
    }
}
