# Contributing

Thank you for your interest in improving Data Structure Advisor.

## Ways to contribute

- Report bugs
- Suggest features
- Improve documentation
- Add or improve data structure explanations
- Improve diagrams or themes
- Refactor code for clarity

## Development setup

Requirements:

- Java 21
- Maven

Run from source:

    git clone https://github.com/FernandoAFOliveira/DS-Tool.git
    cd DS-Tool
    mvn clean javafx:run

## Pull requests

Before submitting a pull request:

1. Make sure the app builds successfully.
2. Test the UI manually.
3. Keep changes focused.
4. Explain what was changed and why.

## Code style

- Prefer clear, readable code.
- Use meaningful names.
- Keep educational content student-friendly.
